# w-GAN

## Overview
I will use a WGAN to create new kinds of bag image. 

## Dependencies (pip install) 
```
cv2
tensorflow
scipy
numpy
```
## Usage
```
cd WGAN
python resize.py
python RGBA2RGB.py
python WGAN.py
```
